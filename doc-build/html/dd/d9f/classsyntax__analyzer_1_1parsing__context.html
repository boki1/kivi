◆ lexer_cursor

const char* syntax_analyzer::parsing_context::lexer_cursor

The location which is used and updated by the re2c module.

NB: The following 2 fields are public, because they are required by bison and re2c in order to keep track of the current location of the concrete process being executed. Because they are publicly visible, they do not follow the standard convention for field names - no "m_" prefix, and can be used without getters.
