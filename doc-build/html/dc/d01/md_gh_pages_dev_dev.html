Building

In order to build the kivi compiler, first checkout the /home/boki/.tmp/kivi/docs/gh-pages/dev/requirements.md "requirements page". After that, in order run ./tools/build.sh to actually build the binary. Having done that, you are ready to go - either ./tools/run.sg source_file.kv to compile your source file or ./tools/test.sh in order to run the test cases for the project.
