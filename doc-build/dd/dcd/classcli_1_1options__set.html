◆ recognize_compil_stage()

void cli::options_set::recognize_compil_stage ( std::string &&  arg)
private

@briev After recognising there is a compile-to stage, recognize_compil_stage gets the exact stage and sets it using the set_compile_to.

Parameters
arg- the argument to be recognised
Note
The argument is the part after the equals sign. For instance, instead of --compile-to=ASM, it looks only for the ASM part.
The argument is valid regardless of the capitalization (ASM and asm both will work, however AsM will not be recognized as a valid command)
